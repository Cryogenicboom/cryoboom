# Cryogenic Boom

A static personal site for GitHub Pages with a retro UNIX-inspired presentation.

## Files

- `index.html` - landing page
- `blog.html` - minimal blog page
- `post-template.html` - reserved template page
- `style.css` - shared styling
- `script.js` - small shared script

## Notes

- The avatar is expected at `avatar.jpg` in the repository root.
- The site uses no build tools or frameworks.
- Project cards open their GitHub repositories in new tabs.

## GitHub Pages

Deploy the repository root directly to GitHub Pages.
